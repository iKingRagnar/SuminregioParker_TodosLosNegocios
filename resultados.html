ℹ️ ¿Por qué Ventas Netas = Utilidad Bruta?
Porque Costo de Ventas no está capturado en la base de datos del módulo de Ventas de Microsip. La jerarquía real de un Estado de Resultados es:
Ventas Netas
  − Costo de Ventas     → Utilidad Bruta
  − Gastos Operativos   → Utilidad de Operación (EBIT)
  − Impuestos / ISR     → Utilidad Neta
Para ver Utilidad Neta real se necesita integrar el módulo de Contabilidad de Microsip (POLIZAS_CONT / CUENTAS_CONT / PRESUPUESTOS). Actualmente este tablero solo tiene acceso al módulo de Ventas y CxC.
